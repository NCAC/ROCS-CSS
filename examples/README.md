# ROCS Examples

Collection d'exemples de composants suivant la méthodologie ROCS.

## Composants disponibles

- [ProductCard](./ProductCard/) — Carte produit e-commerce
- [NavigationMain](./NavigationMain/) — Barre de navigation responsive
- [CheckoutForm](./CheckoutForm/) — Formulaire de paiement multi-étapes

## Structure d'un exemple

Chaque composant contient :
- `index.html` — Structure HTML
- `styles.scss` — Styles ROCS
- `README.md` — Documentation

## Conventions

Tous les exemples suivent strictement :
- 1 classe par élément
- PascalCase pour les Blocs
- `data-*` pour les variantes
- Mixins pour les helpers
